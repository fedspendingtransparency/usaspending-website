Prime recipients of federal contracts or grants are required to file
a subaward report for any subaward greater than or equal to $30,000.

For more information on subaward reporting requirements, please refer to
 <a target="_blank" rel="noopener noreferrer"  href="https://www.sam.gov/fsrs">this informational page</a>
 on <a target="_blank" rel="noopener noreferrer"  href="https://www.sam.gov">SAM.gov</a>, which provides subaward data to USAspending.gov.
