File F (Subaward Attributes) contains data about subawards. These
data are submitted by prime award recipients to 
 <a target="_blank" rel="noopener noreferrer" href="https://www.sam.gov/">SAM.gov</a>.
 New data come to USAspending.gov by the end of the month following the month when the subaward was made. See the entry on <AboutTheDataLink slug="frequency-of-updates-to-subaward-data">Frequency of Updates to Subaward Data</AboutTheDataLink>.
